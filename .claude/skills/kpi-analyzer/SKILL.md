# KPI Analyzer Skill

## 역할
MTBF/MTTR 수치를 제조업 설비 신뢰성 기준으로 해석하고 임원 보고용 인사이트를 생성합니다.

## 평가 기준

### MTBF 수준 판정
```
> 200h  → 🟢 우수: 설비 신뢰성 매우 높음
100~200h → 🟡 양호: 정상 운영 중
50~100h  → 🟠 주의: 고장 빈도 증가 추세
< 50h   → 🔴 경고: 즉각 점검 필요
null    → ⚪ 무고장: 이 기간 정지 없음
```

### MTTR 수준 판정
```
< 1h  → 🟢 우수: 신속 복구
1~3h  → 🟡 양호: 표준 복구
> 3h  → 🔴 주의: 복구 지연
```

### 유지보수 성숙도
```
Preventive > 60% → Preventive 중심 (안정적)
Reactive > 60%   → Reactive 중심 (개선 필요)
40~60%           → 전환 진행 중
```

## 분석 프로세스
1. API에서 데이터 수집
2. null 값 필터링 (무고장 기간 식별)
3. 수준 판정 적용
4. 추세 분석 (전년 대비)
5. 임원 보고 형식 출력

## 출력 템플릿
```
## [공장/공정] 설비 신뢰성 현황

### KPI 수치
| 지표 | 값 | 수준 |
|------|----|------|
| MTBF | Xh | [수준] |
| MTTR | Xh | [수준] |

### 핵심 메시지
[판정 메시지]

### 권고 사항
1. [구체적 행동]
```
